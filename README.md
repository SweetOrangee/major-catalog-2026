# 本科专业目录 · 2026

《普通高等学校本科专业目录（2026 年）》的网页查阅版本，面向同学们在平板和电脑上浏览、检索。

- 13 个学科门类
- 92 个专业类
- 883 个本科专业（含特设专业 `T`、国家控制布点专业 `K`）

## 怎么用

- 在线访问：<https://sweetorangee.github.io/major-catalog-2026/>
- 离线使用：下载 [`dist/offline.html`](dist/offline.html)，双击即可在浏览器中打开，不需要联网。

页面上你可以：

- 按学科门类逐级浏览专业类与本科专业。
- 在顶部搜索框直接输入专业名称、专业代码或类别关键词（按 `/` 可快速聚焦）。
- 点击任意专业查看代码、所属、官方括注等详细信息。

## 数据来源

数据来自中华人民共和国教育部 2026 年 4 月发布的《普通高等学校本科专业目录（2026 年）》。具体规则（基本专业、特设专业、国家控制布点专业、学位授予方式等）以教育部官方文件为准。

页面里的「凡例」入口收录了官方说明的五项要点。

## 目录结构

